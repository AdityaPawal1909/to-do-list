# 📝 To-Do List Application

A simple and responsive To-Do List application built using **HTML, CSS, Bootstrap, and JavaScript**. It helps users organize daily tasks with an intuitive interface and Local Storage support.

## ✨ Features

- ➕ Add Tasks
- ✏️ Edit Tasks
- ✅ Mark Tasks as Complete/Undo
- 🗑️ Delete Tasks
- ⌨️ Add Tasks using Enter Key
- 🔍 Search Tasks
- 📂 Filter (All, Completed, Pending)
- 🔢 Task Counter
- 📅 Date & Time Added
- 🌙 Dark Mode
- 💾 Local Storage Support

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- Local Storage API

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/AdityaPawal1909/To-Do-List.git
   ```

2. Open the project folder.

3. Run `index.html` in your browser.
